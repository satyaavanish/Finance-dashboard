## 🎯 Overview

This Finance Dashboard is a full-featured web application that allows users to track their financial activities, visualize spending patterns, manage budgets, and gain valuable insights into their financial health. Built as part of a Frontend Developer Intern assignment for Zorvyn FinTech, this project showcases modern React development practices with a focus on clean UI, responsive design, and intuitive user experience.

## ✨ Features

### Core Features
- ✅ **Dashboard Overview** - Summary cards for Total Balance, Income, and Expenses
- ✅ **Data Visualizations** - Balance trend charts (Line/Area/Bar) and spending breakdown pie charts
- ✅ **Transaction Management** - Complete CRUD operations with search, filter, and sort
- ✅ **Role-Based UI** - Admin (full access) and Viewer (read-only) modes
- ✅ **Insights Dashboard** - Smart observations about spending patterns
- ✅ **Budget Planning** - Set and track spending limits by category
- ✅ **Advanced Analytics** - Deep insights with multiple chart types
- ✅ **Report Generation** - Export financial data as JSON/CSV
- ✅ **State Management** - Zustand with localStorage persistence

### UI/UX Features
- 🌙 **Dark Mode** - Toggle between light and dark themes with persistence
- 📱 **Fully Responsive** - Works seamlessly on mobile, tablet, and desktop
- ✨ **Smooth Animations** - Engaging transitions and hover effects
- 🔔 **Toast Notifications** - Real-time feedback for user actions
- 🎨 **Glassmorphism Design** - Modern frosted glass aesthetic
- ♿ **Accessibility** - Semantic HTML and keyboard navigation

### Data Features
- 🔍 **Advanced Filtering** - Search by description/category
- 📊 **Multiple Chart Types** - Line, Area, Bar, Pie, Radar, Composed charts
- 📈 **Financial Health Score** - AI-powered insights and recommendations
- 💾 **Data Persistence** - All data saved in localStorage
- 📤 **Export Functionality** - Download transactions as CSV and reports as JSON
